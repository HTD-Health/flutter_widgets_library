# Flutter widgets library

## Getting started

To add a widget to your app:

1. Add this to your `pubspec.yaml` file.

    ```yaml
    dependencies:
      fade_button:
        git:
          url: git@github.com:HTD-Health/flutter_widgets_library.git
          path: packages/fade_button
    ```
