## Getting started

To add a widget to your app:

1. Add this to your `pubspec.yaml` file.

    ```yaml
    dependencies:
      widgets_library:
        git: git@github.com:HTD-Health/flutter_widgets_library.git
    ```
2. You are ready! ðŸ˜‰
